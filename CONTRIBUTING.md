A production-ready NestJS module for Cloudflare R2 object storage management.

## Vision

To provide the NestJS community with a robust, well-documented, and maintainable solution for integrating Cloudflare R2 storage into NestJS applications. The goal is to make file handling simple, type-safe, and production-ready.

---

## How Contributors Can Help

### Current Features
- Signed upload/download URLs
- File CRUD operations
- Nested and array field support
- Storage usage tracking
- Full lifecycle management (create, update, delete)

### Areas for Contribution

1. **New Features**
   - Multi-bucket support
   - File compression before upload
   - Image transformation/resizing
   - Batch operations
   - Progress tracking for uploads

2. **Improvements**
   - Unit test coverage
   - TypeScript strict mode support
   - Error handling enhancements
   - Performance optimizations

3. **Documentation**
   - API reference improvements
   - Example projects
   - Migration guides

4. **Testing**
   - Integration tests with real R2 instance
   - Mock-based testing utilities

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Commit your changes (`git commit -m 'Add amazing feature'`)
4. Push to the branch (`git push origin feature/amazing-feature`)
5. Open a Pull Request

### Guidelines
- Follow existing code style and conventions
- Add tests for new features
- Update documentation accordingly
- Ensure `npm run build` passes
